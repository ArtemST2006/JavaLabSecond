Trie — Программа для работы с префиксным деревом

## Команды
insert <слово> — добавить слово в дерево

contains <слово> — проверить, содержится ли слово

startsWith <префикс> — проверить, есть ли слова с префиксом

getByPrefix <префикс> — вывести все слова с заданным префиксом

all — вывести все слова в дереве

delete <слово> — удалить слово из дерева (с очисткой неиспользуемых узлов)

print — отобразить структуру дерева в виде иерархии

load <файл> — загрузить слова из файла (по одному на строку)

save <файл> — сохранить все слова в файл (по одному на строку)

info — показать справку

stop — завершить программу
 ---
## Формат файлов
Входной файл (для load): текстовый файл, каждая строка — одно слово. Пустые строки игнорируются.
Выходной файл (для save): создаётся заново, содержит все слова из дерева, по одному на строку.
---
## Особенности реализации
Используется кастомный связный список MyList<T> вместо стандартных коллекций.
Узлы дерева удаляются только если они не используются другими словами.
Вывод дерева показывает:
[*] — узел, завершающий слово
